Word Scramble!

by: Devin Moorjani

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I wanted to make a game that was both unique and challenging. I enjoy playing word games, so I decided to create a word scramble game with many different levels and words to unscramble.

2. What I Learned

I fixed a bug that deleted level 1 from the panel after another level was clicked. I also received feedback on the names of the buttons, so I renamed them to make it easier for players to understand.

3. Architecture of My Game

My game has a main class titled 'GameHome', which houses the majority of the code. I also have a runner class and eight level classes; one for each of the levels. All of the level classes extend a main level class, which extends the main game home.

4. Technical Implementation

One aspect of my game that I'm particularly proud of is the 'RETURN' button. While it is a simplistic feature, it is very helpful to the player and it is one of my favorite parts of the game.

5. Most Challenging Part

The most challenging part of this project was connecting the game home with the individual levels. I had to create separate level classes which extended a main level class. This allowed me to easily modify a single level and add more if I wanted to.

6. Future Fixes and Updates

If I were to continue working on this game, I would organize the buttons on the panel to be a little neater. I would also add a 'HINT' feature, which would reveal the first letter of any word. This feature would be limited to three hints per game, so players would have to decide which words to use them on. Finally, I would increase the minimum amount of points to win and add a few more levels, increasing the variety of topics within the game.
